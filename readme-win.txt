WINDOWS Readme

Full recovery not yet supported by windows, backup is supported although limited



Instructions
############

install python-2.7.3 to default location of c:\Python27 first before running backup programs

click start menu and type cmd
you should now have a dos prompt
type
cd \locationofbrixfix 
example, i've extracted brixfix to my desktop

"
Microsoft Windows [Version 6.1.7600]
Copyright (c) 2009 Microsoft Corporation.  All rights reserved.

C:\Users\darkspr1te-windows>cd Desktop\Brickfix-2-1
C:\Users\darkspr1te-windows\Desktop\Brixfix-2-1>backup
"

Rename folder called back to model name , example SHV-E160L-16GB (include emmc size, very important)
create a zip of backup folder and post to sendspace, then post the link to my brixfix thread

www.sendspace.com
